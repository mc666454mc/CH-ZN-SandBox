---

## 📌 项目简介
CH-ZN-SandBox 是一款基于 **Python + PyQt6** 开发的本地代码风险审查工具，可一键扫描、标记并清理代码中的高危内容，全程不上传任何文件，完全保护你的代码隐私。

---

## ✨ 核心功能
- 一键风险扫描：自动识别高危代码并高亮标记
- 精准代码删除：按行号批量删除指定内容
- 可视化编辑预览：内置编辑器，直接修改并保存文件
- 文件快速操作：一键打开、保存、清空编辑内容
- 纯本地运行：无需联网，不上传任何数据

---

## 🚀 一键启动（复制下面指令，直接运行）

### 1. 安装依赖（适配 PyQt6 v6.11.0，国内加速）
```bash
pip install PyQt6==6.11.0 -i https://pypi.tuna.tsinghua.edu.cn/simple
2. 运行主程序
python sandbox.py
🛠️ 使用步骤

1. 点击「打开代码文件」，导入目标文件

2. 点击「一键风险扫描」，查看标记的高危代码

3. 按格式填写行号（如 2,5,9,16），点击「删除指定代码行」

4. 点击「保存修改文件」，导出清理后的文件
⚠️ 注意事项

• 工具仅为辅助审查，无法保证识别所有恶意代码，请谨慎使用

• 删除操作前请备份原始文件，避免误删导致数据丢失

• 项目采用 GPLv3 协议开源，修改后的衍生作品需保持相同协议开源
